To setup page builder please follow below steps.


- Install laravel using below command.
	
- Open command prompt.
	
- Go to xampp/htdocs and execute below command.
	
- composer create-project --prefer-dist laravel/laravel:^7.0 <project_name>
	
- <project_name> Inter your project name for example if your project is abc it will look like below.
	
- composer create-project --prefer-dist laravel/laravel:^7.0 abc


- Some process will happen

- After complete this process go to your project folder using below command.
	
- cd <project_name>
	
- If project name is abc then command will look like below.
	
- cd abc
	
- press enter


- Now execute below commands
	
- composer require hansschouten/laravel-pagebuilder
	
- php artisan vendor:publish --provider="HansSchouten\LaravelPageBuilder\ServiceProvider" --tag=config


- Now before go to next step update yor database configuration, follow below steps
	
- Open .env file.
	
- Update database detail
		
- DB_HOST
		
- DB_PORT
		
- DB_DATABASE
		
- DB_USERNAME
		
- DB_PASSWORD


- APP_URL = http://127.0.0.1:8000/ or http://localhost:8000/admin
	
- Now migrate database using below command
	
- php artisan migrate


- After migration create theme using below command.
	
- php artisan pagebuilder:create-theme <theme_name>


- Now run project using below command
	
- php artisan serve


- Open admin using below URL.
	
- http://localhost:8000/admin
	(if you set app url in env file )
- OR
	
- http://127.0.0.1:8000/admin
 (if you set app url in env file )
